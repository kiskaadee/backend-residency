# ADR 0001: Language Choice (Python)

## Status
Accepted

## Context
We need a backend programming language that optimizes the learning curve and maximizes the likelihood of getting hired quickly. Andres already has familiarity with Python and has explored FastAPI before.

## Decision
Use **Python** as the sole language for this residency.

## Consequences & Trade-offs
- **Pros:** Fast development cycle, lower syntax overhead, Andres already knows it.
- **Cons:** Some enterprise roles prefer Java/C#, but Python still has high demand and allows focusing on core backend concepts instead of syntax.
